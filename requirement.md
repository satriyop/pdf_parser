# PDF Parser

Able to parse PDF files, with high accuracy then provide user editable csv/xls files.

## Objectives
- parse pdf
- output to csv
- output to xls
- output to google sheets (google drive)


## Use case & Roadmap
### 1. single PDF to single CSV
- able to parse PDF file and output csv file with high accuracy.
- should the high accuracy is not fulfilled it should be transparent & informatives to user.

### 2. single PDF to single Excel file
- able to parse PDF files and output XLXS (excel) file with high accuracy.
- might incorporate first roadmap if necessary (if more reliable than to write directly to excel) XLS file.


### 3. multi files/batch PDF to CSV
- able to append safely

### 4. multi files/batch PDF to Excel file
- able to create per sheet per file

### 5. able to ingest PDF files from google drive
- able to get PDF files from google drive to be parse

### 6. output to Google Drive (sheets)
- able to parse pdf
- output can be directly store in google drive (sheets) 

### 7. UI
- cli
- web
